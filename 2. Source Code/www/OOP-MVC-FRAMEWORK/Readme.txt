- Mục tiêu:

	+ Biết cách hoạt động của mô hình MVC.
	+ Nâng cao kĩ năng lập trình PHP.

- Nguyên tắc:
	+ Code 'đẹp'.
	+ Code đúng theo Sequence Diagram MVC. (1)
	+ Đặt tên:
		* class: Viết hoa chữ cái đầu.
		* method: viết thường chữ cái đầu đối với động từ.
		* đặt tên class, biến, hàm rõ nghĩa.

- Xem cách làm việc của MVC ở đây:
http://php-html.net/tutorials/model-view-controller-in-php/ (1)

- Mình sẽ code ở Backend, vì chỉ làm trang admin.

- Mô tả các folder: 
	+ Asset: lưu các file css, js, images.
		* css: lưu file custom.css.
		* js: lưu các file customjs.js.
	+ Config: lưu file config database.
	+ Libs: lưu các thư viện ( Chứa các hàm hoặc class ).
	+ Controller: lưu các Controller.
	+ Model: lưu các model.
	+ View: lưu các view.
	+ Test: lưu các file test debug.

- Phân chia công việc:
